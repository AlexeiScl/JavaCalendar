# JavaCalendar
This project uses java to make a Calendar, with a working UI, using a JFrame, and functioning memory storage with easy - to - modify code.
To run this code: Download the calenr project file, open Eclipse, and put the file into the directory. Now, in eclipse, use the file - open - tab and then open this file.
When the file is finally opened, you can ciew the app and run it.
